## Authentication

All API requests require authentication using an API Key.

You must use a valid API Key to send requests to the API endpoints. To obtain an API key, please contact the Reboxed team directly.

## Important Notes

- The API only supports HTTPS communications
- Any HTTP requests will receive a 301 redirect to the corresponding HTTPS resource
- Responses are returned in JSON format
- Error responses include an `error` key in the JSON response
